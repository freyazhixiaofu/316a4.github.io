How Height Changes Across the Life Course

Ruler: Draw a rectangle on the chart, then drag it.

Each vertical stack shows the height distribution for an age bin. Childhood uses finer age bins where growth is rapid (yearly under 10, every 2 years from 10–20), and adulthood uses coarser bins (every 5 years above 20) to reduce clutter as changes slow down.

For each age, thin vertical rectangles show how common each height is. Taller rectangles mean a higher percentage of people at that height. Switch genders above to approximate a life-course profile by comparing cross-sectional height distributions.

Data: NHANES Body Measures (BMX_L) and Demographics (DEMO_L) processed locally into docs/data/height_distributions_by_age_gender.json.
